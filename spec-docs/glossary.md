# 用語定義

キャンプギアドメインおよび本プロジェクト固有の用語。

## ドメイン用語

| 用語 | 定義 |
| ---- | ---- |
| ギア (Gear) | キャンプ用品の総称。テント・タープ・チェア・ランタン等 |
| レイアウト (Layout) | ユーザーが投稿するキャンプサイトの写真 + 使用ギアの配置情報 |
| イメージマップ (Image Map) | レイアウト画像上に配置された商品タグの位置情報 |
| タグポジション (Tag Position) | イメージマップ上の個々のタグの座標（x, y） |
| カテゴリ (Category) | 商品の大分類（テント、タープ、チェア、テーブル等） |
| サブカテゴリ (Sub Category) | 商品の小分類（ドーム型、ワンポール型、ツールーム型等） |
| ブランド (Brand) | メーカー・ブランド（スノーピーク、コールマン、DOD等） |
| インベントリ (Inventory) | ユーザーが所有するギアのリスト（持ち物リスト） |

## システム用語

| 用語 | 定義 |
| ---- | ---- |
| Server Components (RSC) | React Server Components。サーバー側でレンダリングされるコンポーネント |
| Server Actions | フォーム送信等のサーバー側処理を関数として定義する Next.js の機能 |
| ISR | Incremental Static Regeneration。静的ページをバックグラウンドで再生成 |
| Presigned URL | 一時的なアクセス権を付与した S3 の署名付き URL |
